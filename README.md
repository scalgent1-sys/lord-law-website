# LORD Law Website

Static GitHub Pages rebuild for Lagmay Olavere Rosas & Domingo Law Office.

## Files

- `index.html` - single-page website with sections for home, firm profile, practice areas, team, book, and contact
- `assets/css/styles.css` - responsive visual system and layout styles
- `assets/js/main.js` - mobile menu, active nav, reveal effects, and static contact form behavior
- `assets/images/` - local imagery downloaded from the existing LORDLAW website

## Run Locally

Open `index.html` in a browser, or serve the folder with Node:

```powershell
node server.js
```

Then visit `http://localhost:8080`.

## Deploy On GitHub Pages

1. Push this folder to a GitHub repository.
2. In GitHub, open **Settings > Pages**.
3. Select the branch and root folder.
4. Save the Pages setting.

The site is intentionally static, so no build step is required.
